# Pakistani Tax Advisor - Frontend

React frontend application for calculating Pakistani income tax for 2024-25.

## Features

- 🇵🇰 Pakistan-specific tax calculations
- 📊 2024-25 tax slabs implementation
- 🎯 User-friendly tax calculator interface
- 📱 Responsive design

## Getting Started

### Prerequisites

- Node.js (v14 or higher)
- npm or yarn

### Installation

```bash
# Install dependencies
npm install

# Start development server
npm start

# Build for production
npm run build
```

## Available Scripts

- `npm start` - Runs the app in development mode
- `npm run build` - Builds the app for production
- `npm test` - Launches the test runner
- `npm run eject` - Ejects from Create React App

## Tech Stack

- React 18
- React Router DOM
- Axios for API calls
- CSS3 for styling

## Tax Slabs Implemented

- Salaried Individual tax slabs (2024-25)
- Non-salaried Individual tax slabs (2024-25)
- Filer vs Non-filer calculations
